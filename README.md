# uniapp-request-sdk
这是一款具备`通用性`、`简洁性`、`扩展性`的uni-app请求封装的工具包



## 目录结构
```txt
└── utils 
    ├── api // 放置请求示例文件
    │   └── index.ts 
    ├── loading.ts // loading 相关
    ├── platform.ts // 平台相关
    ├── request.ts // 核心代码
    └── type 
        └── index.ts // 定义的一些关于请求的类型, 建议放在 global.d.ts 全局 ts 类型定义中
```
